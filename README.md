# JapAssist Frontend

日本語学習支援アプリケーションのフロントエンド

## 機能

- 文法チェッカー
- 文章アシスタント
- 単語検索
- 学習ボード

## 技術スタック

- React
- Material-UI
- Tailwind CSS
- Google Cloud Platform

## CI/CD

GitHub Actionsを使用して、メインブランチへのプッシュ時に自動的にGoogle App Engineにデプロイされます。
